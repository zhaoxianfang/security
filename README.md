# Laravel Security Middleware

高级安全拦截中间件包，为Laravel应用提供全面的安全防护。

## 功能特性

- 🔒 **多层安全检测** - 请求内容、URL路径、User-Agent等多维度检测
- ⚡ **性能优化** - 智能缓存、预编译正则、分层检测策略
- 🛡️ **IP管理** - 白名单、黑名单、动态封禁管理
- 📊 **速率限制** - 分钟、小时、天级别多层限制
- 🎯 **威胁检测** - SQL注入、XSS、命令注入等攻击检测
- 📱 **响应式界面** - 美观的拦截页面，支持深色模式
- 🔧 **高度可配置** - 支持动态配置源和自定义规则

## 安装

```bash
composer require zxf/laravel-security-middleware
```
