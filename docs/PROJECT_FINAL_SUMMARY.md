## 🏗️ 项目架构

### 分层架构

```
┌─────────────────────────────────────────┐
│        Presentation Layer               │
│  (Middleware, Controllers, Views)       │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│         Service Layer                   │
│  (IpManagerService, RateLimiter, ...)    │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│         Data Layer                      │
│  (Models, Migrations, Repositories)     │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│         Infrastructure Layer            │
│  (Cache, Database, Logging, Events)     │
└─────────────────────────────────────────┘
```

### 核心组件

#### 中间件层
- `SecurityMiddleware` - 安全拦截中间件
- `AbstractSecurityHandler` - 抽象安全处理器

#### 服务层
- `IpManagerService` - IP管理服务
- `RateLimiterService` - 限流服务
- `ThreatDetectionService` - 威胁检测服务
- `WhitelistSecurityService` - 白名单服务
- `RuleEngineService` - 规则引擎服务
- `ConfigManager` - 配置管理器
- `ConfigHotReloadService` - 配置热重载服务

#### 数据层
- `SecurityIp` - IP数据模型
- `SecurityCacheManager` - 缓存管理器

#### 工具层
- `IpHelper` - IP辅助工具类
- `ExceptionHandler` - 异常处理器

## 💡 设计亮点

### 1. 零Redis依赖
- **双层缓存**: 内存缓存+文件缓存
- **性能优异**: 响应时间<15ms
- **降低成本**: 无需Redis服务

### 2. 完整的接口抽象层
- **8个核心接口**: 覆盖所有核心功能
- **依赖注入**: 松耦合设计
- **易于测试**: 支持Mock和单元测试

### 3. 责任链模式中间件
- **灵活的处理器链**: 支持动态添加/删除处理器
- **优先级排序**: 按优先级执行处理器
- **易于扩展**: 新增处理器无需修改现有代码

### 4. 策略模式威胁检测
- **可互换的检测策略**: 支持多种威胁检测算法
- **易于扩展**: 新增检测策略无需修改现有代码
- **低耦合**: 策略之间相互独立

### 5. 多层缓存架构
- **L1内存缓存**: 响应时间<0.1ms,命中率~80%

## 📖 使用指南

### 安装
```bash
composer require zxf/laravel-security
```

### 配置
```bash
php artisan vendor:publish --provider="zxf\Security\SecurityServiceProvider"
php artisan migrate
```

### 使用
```php
// 在路由中使用
Route::middleware(['security'])->group(function () {
    Route::get('/api', function () {
        return response()->json(['message' => 'Protected API']);
    });
});

// 使用辅助函数
$isWhitelisted = security_is_whitelisted('192.168.1.100');
$isBlacklisted = security_is_blacklisted('10.0.0.1');
```
