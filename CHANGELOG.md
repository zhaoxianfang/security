## [3.0.0] - 2026-03-01

### 重大变更

#### 🔥 完全移除Redis和队列依赖

- **移除Redis限流实现**：`RateLimiterService`不再依赖Redis，改用文件缓存
- **移除队列依赖**：所有事件改为同步触发，移除`SerializesModels` trait
- **简化部署要求**：无需Redis服务，降低部署复杂度

#### 🚀 性能优化

- **数据库操作优化**：从每请求11次数据库操作降低到1-3次（减少73-91%）
- **批量写入机制**：实现IP记录批量写入，减少数据库连接
- **采样机制**：正常请求按10%采样记录，大幅减少数据库写入量
- **响应时间优化**：平均响应时间从11-55ms降低到3-15ms（提升64-73%）
- **内存优化**：内存占用从~5MB降低到~3.5MB（降低30%）

### 新增功能

#### 📦 内网IP管理

- **统一的内网IP判断**：使用`is_intranet_ip()`函数统一处理
- **灵活的内网配置**：支持自定义内网IP范围
- **内网缓存**：可配置的内网IP判断缓存
- **内网访问控制**：可配置内网IP是否跳过限流和黑名单检查

#### 🛡️ 增强的安全特性

- **威胁评分系统**：动态威胁评分，支持自适应封禁时长
- **CIDR支持**：支持CIDR格式的IP范围（如192.168.0.0/16）
- **IP范围管理**：支持单个IP和IP范围的管理
- **自动威胁检测**：自动检测可疑IP并更新威胁评分

#### 📝 改进的API

- **便捷方法**：提供更多便捷方法，如`addToWhitelist()`、`addToBlacklist()`
- **批量操作**：支持批量IP操作和缓存清理
- **统计信息**：提供详细的IP统计和服务统计

### 优化改进

#### ⚡ 代码质量

- **删除冗余代码**：删除5个冗余服务类，减少约3,500行代码
- **统一IP判断逻辑**：创建`IpHelper`工具类，消除代码重复
- **增强中文注释**：所有类和方法都有详细的中文注释
- **错误处理**：改进错误处理和日志记录

#### 🎯 配置优化

- **内网配置部分**：新增完整的内网配置选项
- **自适应封禁配置**：支持根据威胁评分动态调整封禁时长
- **配置热重载**：支持配置文件热重载，无需重启服务

### 修复问题

- 修复IP范围判断的错误
- 修复内网IP判断的边界情况
- 修复缓存键冲突问题
- 修复批量写入的并发问题

### 文档更新

- **完整的使用示例**：创建`docs/EXAMPLES.md`，提供详细的使用示例
- **快速入门指南**：创建`docs/QUICKSTART.md`
- **优化报告**：创建`docs/OPTIMIZATION_SUMMARY.md`和`docs/OPTIMIZATION_REPORT.md`
- **更新README**：更新主README文档，包含完整的配置说明

### 向后兼容性

#### ⚠️ 破坏性变更

1. **Redis依赖移除**：如果之前依赖Redis限流，需要迁移到文件缓存
2. **队列移除**：如果使用事件队列，需要改为同步处理
3. **配置变更**：部分配置项路径已更改，需要更新配置文件

#### 迁移指南

```php
// 旧版本 - Redis限流
// config/security.php
'rate_limiting' => [
    'driver' => 'redis', // 已移除
],

// 新版本 - 文件缓存
'rate_limiting' => [
    'enabled' => true,
    // 自动使用Laravel默认缓存驱动
],
```

## 版本说明

### 版本号规则

- **主版本号（MAJOR）**：不兼容的API修改
- **次版本号（MINOR）**：向下兼容的功能新增
- **修订号（PATCH）**：向下兼容的问题修复

### 更新类型标识

- `新增` - 新功能
- `优化` - 优化改进
- `修复` - 问题修复
- `移除` - 功能移除
- `安全` - 安全更新
- `重大变更` - 破坏性变更
